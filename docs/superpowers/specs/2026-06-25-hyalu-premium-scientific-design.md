# Hyalu Brasil Premium Scientific Design

## Goal

Upgrade the existing Hyalu Brasil one-page website into a more premium, responsive, scientific and technological experience while preserving the approved B2B corporate structure and copy.

## Visual Direction

The site should feel like a high-trust scientific and healthcare technology company: polished, precise, regulated, and globally connected. Use a refined palette of deep navy, clinical white, electric blue, cyan, and subtle green accents. The atmosphere should reference laboratory science, molecular structures, global import logistics, and strategic healthcare supply.

## Layout

Keep the same section order: header, hero, about, portfolio and solutions, differentials, corporate contact, footer. Increase visual sophistication through layered backgrounds, precise spacing, premium cards, glass-like panels, and a stronger hero composition. The page must remain easy to read and clearly focused on conversion.

## Motion

Add tasteful animations: header elevation on scroll, hero reveal, subtle floating hero media, molecular/connection background motion, card reveal on scroll, hover transitions, and form/button feedback. Respect `prefers-reduced-motion`.

## Responsiveness

The site must remain fully responsive on mobile, tablet, and desktop. Mobile should feel intentionally designed, not merely stacked: clear header, large readable hero, full-width CTAs, properly cropped image, and no horizontal overflow.

## Verification

Validate required content with the smoke test. Capture desktop and mobile screenshots and inspect them for premium appearance, readability, section order, button fit, image loading, and lack of layout overflow.
